After installing Git, run these commands one by one in PowerShell:

1. Initialize git repository:
git init

2. Add all files:
git add .

3. Create first commit:
git commit -m "Initial commit: Secure ArgueIt platform"

4. Add your GitHub repository (replace with your actual URL):
git remote add origin https://github.com/jimmyss/argue.it.git

5. Push to GitHub:
git push -u origin main

If step 5 fails, try:
git branch -M main
git push -u origin main

That's it! Your entire project will be on GitHub.
