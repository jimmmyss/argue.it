# Complete Git Setup Process

## Step 1: Install Git
Download from: https://git-scm.com/download/windows
Restart PowerShell after installation

## Step 2: Configure Your Identity
git config --global user.name "Your Full Name"
git config --global user.email "your.email@example.com"

## Step 3: Verify Configuration
git config --list

## Step 4: Initialize and Push Repository
git init
git add .
git commit -m "Initial commit: Secure ArgueIt platform"
git remote add origin https://github.com/jimmmyss/argue.it.git
git push -u origin main

## Optional: Set Default Branch (if needed)
git config --global init.defaultBranch main

## Troubleshooting:
If push fails, try:
git branch -M main
git push -u origin main

## Check Git Version:
git --version
